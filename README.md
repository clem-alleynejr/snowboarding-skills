# Snowboarding Skills Tracker

This app is used to track your snowboarding progression from beginner to pro!

Create your own customizable list of snowboarding skills you wish to track. Each individual skill has the properties of - skill level, user proficiency level, and any user - that can be viewed and edited at any time.
